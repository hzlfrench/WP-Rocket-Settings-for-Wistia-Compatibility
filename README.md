# WP-Rocket-Settings-for-Wistia-Compatibility

Ensure smooth playback by excluding Wistia's JavaScript from File Optimizations.

WP Rocket's file optimization features can break Wistia embeds if not configured correctly. This guide covers which Wistia scripts to exclude and how to update your WP Rocket exclusion lists.
